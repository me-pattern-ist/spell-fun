# Math Generation Workflow

This workflow regenerates the `assets/math_fractions.csv` file using a local Ollama instance.

1. Run the Python generation script for fractions.
// turbo
`python3 tool/generate_math_problems.py`

2. Run the Python generation script for multiplication.
// turbo
`python3 tool/generate_multiplication_problems.py`
